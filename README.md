# Админ-панель

Веб-приложение админ-панели с компонентной архитектурой.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Структура проекта

- `src/components/` - Переиспользуемые компоненты (Layout, Sidebar, Header, Tabs, Table, Form)
- `src/pages/` - Страницы приложения (Home, Functions, Requisites)
- `src/App.tsx` - Главный компонент с роутингом

## Страницы

- **Главная** (`/`) - Дашборд с таблицей заказов, справочниками и отчетами
- **Функции** (`/functions`) - Список функций экспорта данных
- **Реквизиты** (`/requisites`) - Редактирование реквизитов компании с тремя вкладками:
  - Информация о компании
  - Адреса компании
  - Информация о банке
